# 🎓 Student Performance Tracker & Assistant

This software is a student tracking and performance analysis solution, developed as an academic project. The application utilizes the low-code paradigm through Streamlit and XanoScript for backend.

The main goal is to transform raw performance data into visual and actionable insights, facilitating time management and study goals.

## 🚀 Features

* **Performance Dashboard:** Visual and interactive interface to track student metrics.
* **Goal Assistant:** Integrated AI assistant to guide the student through their objectives and study plans.

## 🛠️ Technologies Used

* **Language:** Python
* **Web Interface (UI/UX):** Streamlit
* **Data Manipulation:** Pandas
* **Back-end & Database:** Xano (No-Code Platform / REST APIs)
* **API Integration:** Requests
* **Development Tools:** Watchdog (for hot-reloading during development)
* **IA tool:** Open Spec

## 📦 Dependencies

The project requires the following Python libraries:
* `pandas`
* `streamlit`
* `watchdog`
* `requests`
